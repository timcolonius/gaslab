GasLab consists of two complementary components.

The GasLab Python package provides a programmatic interface for constructing and analyzing compressible-flow processes, including shocks, Prandtl–Meyer expansions, quasi-one-dimensional area change, Fanno flow, and Rayleigh flow. It is designed for use in scripts and Jupyter notebooks, and supports applications ranging from homework exercises to research calculations.

The GasLab web application, built on the same computational engine, provides an interactive graphical entry point into these problems. Users can construct flow processes step-by-step, inspect the resulting states, view them graphically, and export the equivalent Python commands for further analysis.

A collection of worked examples is available as Jupyter notebooks in the repository. These illustrate typical workflows and standard compressible-flow problems, and provide a natural bridge between the graphical interface and direct use of the Python package.

Available notebooks:
Getting started using GasLab, Regimes of a de Laval nozzle, Oblique shocks and Prandtl-Meyer expansions, Wave interactions.
