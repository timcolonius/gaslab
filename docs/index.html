GasLab
Interactive Compressible Flow Explorer
Launch App View Source on GitHub Getting started


GasLab is a Python package for exploring compressible flow processes, including shocks, Prandtl-Meyer expansions, quasi-one-dimensional area change, Fanno flow, and Rayleigh flow. Users can define flow/thermodynamic states and processes connecting them, and use these to solve problems from homework to research.

There is also a browser-based intercative tool that uses the package to build flow processes graphically, inspect the resulting states, view the graphically, and export the equivalent Python commands for further analysis.
